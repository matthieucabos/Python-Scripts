Mapping.build_ip_mac_dict

Mapping.build_ip_mac_dict(tftp_Content)

Building Ip 2 @Mac dictionnarry from tftp boot server files (connected people). We are getting the full connected Users Mac => IP dictionnary using regular expression :

  • [0-9a-z]{4}.[0-9a-z]{4}.[0-9a-z]{4} : Give us the MAC adress since the tftpboot files

  • ([0-9]/){2}[0-9]* : Give us the Hardware Cisco Port Number since the tftpboot files

  • d+.d+.d+.d+ : Give us the IP Adress since the tftpboot files

Parameters

Type

Description

tftp_Content

string

The tftpboot file raw content
Returns

Dictionnary : The dictionnary with ip/mac correspondance
